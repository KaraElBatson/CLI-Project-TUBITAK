## Node.js Kurulumu

Önce bilgisayarında Node.js kurulu olup olmadığını ve versiyonlarını kontrol et.

Terminali (PowerShell, CMD veya Git Bash) aç ve aşağıdaki komutları sırayla çalıştır:

```bash
node --version
npm --version
```

Eğer bu komutlar versiyon numarası döndürüyorsa (örneğin `v22.3.0` gibi), Node.js ve npm yüklüdür.

Komutlar bulunamadı hatası alırsan, Node.js kurulu değildir ve resmi siteden indirip kurman gerekir:

- `https://nodejs.org` adresine git.
- Windows için LTS (önerilen) sürümü indir.
- Kurulum sihirbazını “Next” diyerek tamamla.

Kurulumdan sonra terminali kapatıp tekrar aç ve komutları yeniden çalıştır:

```bash
node --version
npm --version
```

Versiyon numaralarını görüyorsan kurulum başarıyla tamamlanmış demektir.
